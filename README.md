# 🌐 ポートフォリオサイト

このポートフォリオは **Next.js + TypeScript + MUI** で構築し、
**Vercel + Cloudflare** 経由で無料公開しています。
主に、業務改善システムやDX推進プロジェクトなど、私が実際に設計・開発したアプリケーションを紹介しています。

---

## 🚀 使用技術

| カテゴリ | 技術 |
|-----------|------|
| フロントエンド | Next.js (App Router), React 18, TypeScript |
| UIライブラリ | MUI (v7) |
| スタイリング | MUI Theme + Tailwind CSS |
| ホスティング | Vercel |
| ドメイン | Cloudflare DNS（Xserverで取得したドメインを接続） |
| バージョン管理 | GitHub（公開リポジトリ） |
| その他 | dayjs / react-hook-form / zod / react-toastify |

---

## 💡 主な機能

- モバイル対応のレスポンシブUI
- JSONデータによる **モックCRUD（擬似登録・編集・削除）**
- 操作結果をトースト通知で表示
- Googleログイン風UI（演出のみ）
- 実システム（Laravel / WordPress）はXserver上にリンク掲載
- Supabase等の外部DBなしで完全無料運用
